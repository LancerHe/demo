公司简介

您当前位置：翰廷装饰 >> 基本信息 >> 浏览文章

    陕西翰廷装饰工程有限公司成立于二零零七年，是一家专业工程装修公司，公司具有设计及施工双乙级资质，拥有优秀的家装、工装设计团队和优秀的南方施工队伍，主要承接精装居室、办公空间、酒店、餐饮、娱乐、园林景观及大型连锁经营机构的装饰设计及施工工程，是一家集装饰设计、空间规划、工程管理和标准化施工为一体的专业化公司。
    企业一直致力于西部建设逐渐形成了自己独有的企业文化和艺术风格，总结出了一套适应自身发展需要的战略措施。雄厚的资本、丰富的经验、专业的队伍，成就了今天的翰廷品牌。企业在管理上已卓有成效，形成了一套较为先进的管理理念和经营理念。
    为保证每个项目的工程质量，我公司在长期材料采购中，逐渐形成了自身相对稳定的材料供应渠道，并可为供应的材料质量、数量、品种提供保障和售后服务。稳定、快速的供应渠道，是保证工程质量及顺利进行的一个重要环节。
    我公司组织机构共设置设计部、工程部、技术部、材料部、财务部、售后部五大部门，共有在职管理人员30余人。公司主要领导人及项目经理均从事装饰装修行业十多余年，具有丰富的大型设计经验、施工管理经验以及严格的管理标准。我公司设计团队共有国家注册中级室内设计师2人，国家注册室内设计师4人，其余均毕业于各设计专科学院；施工管理团队现有注册二级建造师6人，长期施工队均为安徽、湖北、四川籍人士，共计100余人。
    自成立来，我们始终坚持设计零缺憾、施工零缺点、客户零投诉的经营原则。公司严格按照《公司法》实施规范管理，近几年先后完成多项设计与施工工程，受到了广大客户的高度认同，市场影响力直线上升，企业知名度不断扩大。
    翰廷人竭诚为您描绘锦绣蓝图，与您共同携手并进，共创美好未来！
